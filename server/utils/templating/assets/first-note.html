Here is little tutorial on how to use Keycap:

  • Every shortcut begins with Ctrl or CMD key (dependent on your OS).

  • Select text you want to edit: make be bold. Now you can press CMD+b shortcut or button with B icon in formatter menu.

  • You can also hover over the button and it will tell you which keys to press to active it (if you use computer).

  • Highlight portion of this text and try toggling the list formatting (second button in formatter). It firstly will toggle on or off the list and then it will start to cycle through all formats: bullet list -> numbered list -> task list -> off

  • Such behavior of toggling off and on and then cycling through available formats also applies to headings

Keycap, can have a lot of styles: bold, italic, code, strike, links.

Our editor supports markdown, so you can use it to seamlessly format your text. When the Esc key is pressed, the editor will loose focus , to bring the focus back, press i (insert)

Global shortcuts:

  • CMD+k - opens search. Type note name then enter and you will be redirected to that note (or folder). It also has a couple of commands. All commands starts with /.

  • CMD+Shift+X - opens left sidebar. From there you can open note details (this also available through CMD+k -> /details) and your recent notes.

  • CMD+Shift+F - opens right sidebar. All your notes and folders live there. This sidebar also contains button for creating new items (folders or notes)
